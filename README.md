COVID-19 Dashboard using Power BI

This is a Power BI dashboard project based on COVID-19 data analysis.  
The dashboard contains two pages:

1. India Dashboard
- Shows COVID-19 cases in India
- State-wise analysis
- Active, recovered and death cases
- Different charts and KPIs for better understanding

2. World Dashboard
- Global COVID-19 analysis
- Country-wise comparison
- Total confirmed, recovered and death cases
- Interactive visualizations

Tools Used
- Power BI
- Excel / CSV dataset

Features
- Interactive dashboard
- Filters and slicers
- Charts and graphs
- Data analysis and visualization

Files Included
- Power BI project file (.pbix)
- Dataset
- Dashboard screenshots

This project helped in understanding data visualization and dashboard creation using Power BI.
